Uma aplicação web simples para gerenciar ganhos e gastos, desenvolvida como projeto prático do curso ReactJS Projects For Beginners.

Funcionalidades:

-Adiciona transações de ganho e gasto
-Remove transações existentes
-Calcula saldo total, ganhos e gastos automaticamente
-Armazenamento de dados no navegador usando LocalStorage
-Interface limpa e responsiva

Tecnologias Utilizadas:

-JavaScript
-React
-HTML5
-CSS3
-LocalStorage
